Python将图片转换为字符串 - CharacterMatrix

2020-04-14 1 min read # 日常 # python

发现了一个有趣的Python项目，将图片转换为字符串，项目地址

基本原理是：将图片转化为灰度图像，再根据强度替换字符串

根据这个项目做了下，使用Pyinstaller打包了，使用方法如下：

CharacterMatrix [图片路径]

会在图片目录下创建一个名为Character Matrix.txt的文件

如果您想要使用其他保存名称,使用方法如下

CharacterMatrix [图片路径] -o [保存名称]

如果您想要保存到其他路径，使用方法如下

CharacterMatrix [图片路径] -o [保存路径]
